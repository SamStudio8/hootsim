#Hootsim
An extensible agent-based model simulator written in C++

##Requirements
####Simple and Fast Multimedia Library v2.0
<a href="http://www.sfml-dev.org/">http://www.sfml-dev.org/</a>

##Controls
* **N** New Hoot
* **R** Reset Stage
* **C**  Screenshot
* **I/O** Zoom In/Out
* **Arrows** Navigate Stage
* **Q** Quit
